# Autocomplete using a Prefix Tree (Trie) implemented in C

## Quick start

### Build

```console
$ make
$ ./trie --help # for usage
```

### Dump the Trie as SVG (Requires [Graphviz](https://graphviz.org/))
```
$ ./trie dot
$ # open ./graphviz/output.svg
```

### Autocomplete prefix

```console
$ ./trie complete <prefix>
```
